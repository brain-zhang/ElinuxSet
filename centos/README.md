setup
=====

Deployment Scripts for CentOS


## Installation

Execute the following commands at your servers command-line:

	$ yum -y install git
	$ git clone git@github.com:memoryboxes/ElinuxSet.git
	$ cd centos

###PPTP

	$ sh centos_pptp_install.sh

###LAMP(Apache)

	$ sh centos_lamp.sh

###LEMP(nginx)

	$ sh centos_lemp.sh

###svn1.8

	$ sh svn1.8_install.sh

###Centos6 setup

    $ sh oh-my-centos6.sh
